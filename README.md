# Поддержка корпоративных сайтов
Поддержка корпоративных сайтов — это комплексная услуга, предоставляемая на основе современных технологий и широкого спектра CMS и фреймворков, включая Cetera CMS, Laravel, Yii2, InSales, 1С-Битрикс, Bitrix24, WordPress, WooCommerce, Ecwid, OpenCart, Drupal, Joomla, Magento2, Shopify, а также самописные системы на PHP и Python. Услуга включает поддержку инфраструктуры, развитие сайта, техническое сопровождение и продвижение в интернете, что позволяет обеспечить стабильную работу и развитие корпоративного ресурса.

## Услуга предоставляется в двух основных тарифах:

### Безлимитное абонентское обслуживание

Этот тариф предлагает фиксированную ежемесячную плату от 65 000₽ (в зависимости от типа проекта), которая покрывает неограниченный объем задач и работ в рамках согласованного мастер-плана. Модель подходит для проектов с высокой динамикой развития, которые требуют регулярных доработок, обновлений и оптимизации.

#### Преимущества:

- Гарантированное SLA и поддержка 24/7
- Включена инфраструктура размещения и разработки
- Закрепленная команда специалистов и минимальные бюрократические издержки
- Быстрое выполнение срочных задач (в том числе «день в день»)
- Выполнение любых изменений без ограничений по объему и согласованиям
- Регулярная прозрачная отчетность и управление проектом
- Возможность работы с любым количеством сайтов заказчика
- Оплата авансом по абонентскому договору с возможной постоплатой

Тариф с оплатой за фактически затраченные часы (1 800 ₽ за час) подходит для проектов с ограниченным объемом задач или нерегулярной поддержкой. Это удобный вариант для разовых доработок, исправлений или небольшой технической поддержки.

#### Особенности:

- Оплата постфактум по фактическому времени, с отчетностью и актами
- Гибкость в планировании бюджета и контроля затрат
- Оценка задач по прайс-листу с возможностью детального согласования
- Подходит для компаний с небольшим или нерегулярным объемом работ
- Отсутствие фиксированной абонентской платы и обязательств по объему
